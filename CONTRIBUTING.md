# Contributing to IWE

This repository is an automatically published English projection of the
working source. It is not a place to open pull requests.

- **Report a bug or request a feature:** open an issue on the source
  repository — [https://github.com/TserenTserenov/FMT-exocortex-template/issues](https://github.com/TserenTserenov/FMT-exocortex-template/issues)
- **Contribute code or docs:** fork [https://github.com/TserenTserenov/FMT-exocortex-template](https://github.com/TserenTserenov/FMT-exocortex-template) (the working source, in
  Russian) and open a pull request there.

See the source repository's [CONTRIBUTING.md](https://github.com/TserenTserenov/FMT-exocortex-template/blob/main/CONTRIBUTING.md)
for the full guide.
