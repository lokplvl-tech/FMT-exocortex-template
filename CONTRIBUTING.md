# Contributing to IWE

- **Report a bug or request a feature:** open an [issue](https://github.com/iwesys/IWE/issues).
- **Questions, ideas, show your setup:** start a [discussion](https://github.com/iwesys/IWE/discussions).

This distribution is assembled by an automated publishing pipeline,
so direct pull requests can't be merged — every accepted change
ships through the pipeline. If you want to change something, open
an issue describing it and we'll take it from there.
